# ML-PREP-2025
 
 ML-PREP is a 2-year project, implemented by [Caraga Stata University](https://www.carsu.edu.ph/) in cooperation with [DOST-PHIVOLCS](https://www.phivolcs.dost.gov.ph/index.php), and funded by [DOST](https://www.dost.gov.ph/). 
 
---
 
## Training 
 
 This repository contains the training material for the ML-PREP project. The training covers 3 topics:
 
 - Basic and advanced training in [Machine Learning](./01MachineLearning)
 - Introduction to Geographical Information Systems (GIS) and [Geospatial Data Analysis](./02GeoDataSceince)
 - Elements of seismic [Wave Propagation](./03WavePropagation)
 
 Each theme contains theoretical material with programming examples and exercises.
 
---

## Research Projects

 The objective of the training is to execute [research projects](./04Projects) whose aim is to deepen and enhance the earthquake induced landslide (EIL) hazard mapping capabilities of PHIVOLCS.
